# Desafios Técnicos – Treinamento Acadêmico

Este repositório reúne exercícios e pequenos programas desenvolvidos durante atividades acadêmicas, com foco no treinamento de **lógica de programação** e no contato inicial com **C#** e **C++**.

Os projetos aqui presentes foram revisados e ajustados com o objetivo de compor um **portfólio acadêmico**, mantendo uma abordagem simples, porém tecnicamente correta e organizada.

---

## Objetivo

* Praticar conceitos fundamentais de programação
* Exercitar lógica condicional, entrada e saída de dados
* Consolidar a base em linguagens de uso comum no mercado
* Demonstrar evolução e capacidade de revisão de código

---

## Tecnologias utilizadas

* **C#** (aplicações de console)
* **C++** (programas introdutórios)
* Git e GitHub para controle de versão

---

## Estrutura dos códigos

Os códigos foram mantidos propositalmente simples, priorizando:

* Clareza da lógica
* Legibilidade
* Uso consciente de recursos da linguagem
* Correções de erros técnicos comuns em projetos iniciais

Alguns exemplos de melhorias aplicadas:

* Remoção de bibliotecas não utilizadas
* Ajustes de tipos de dados
* Validação básica de entradas do usuário
* Padronização de mensagens e organização do código

---

## Observação

Este repositório tem caráter **educacional**. Ele não busca apresentar soluções avançadas, mas sim demonstrar fundamentos sólidos, boa organização e postura profissional compatível com o início da carreira em desenvolvimento de software.
